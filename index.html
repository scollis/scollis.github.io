The Variant Trio template

Inspired by the Variant Duo template, here is Variant Trio. It gives you the option to choose between two or three columns with a sidebar menu. This is a completely free template, and it may be used without any limitations or obligations. If you decide to use this template to build a website or a blog/CMS theme, please leave the design credits in the footer.

Sample image

Two or three columns

To use two columns, put your main content inside a div with the class name "full". To use three columns, make one div with the class name "left" and another with the class name "right" and then place a hr with the class name "clear" after both divs.
